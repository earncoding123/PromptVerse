# **App Name**: PromptVerse

## Core Features:

- User Authentication: Allow users to sign up and sign in using Google or email, managed by Firebase Auth.
- Prompt Browsing: Display a list of prompts with title, description, tags, category, and cover image.
- Prompt Detail View: Show the full prompt content, price, and download options (free or gated).
- Admin Panel: Provide a secure admin interface for creating, editing, deleting, and publishing prompts, including image upload and SEO field management.
- SEO Optimization Tool: Allow the admin user to evaluate their content using SEO principles, with suggestions for improving content before it goes live. This is an LLM-powered tool that incorporates various best practices.
- User Dashboard: Enable users to view their purchased prompts, downloads, and manage their profile.
- Sitemap Generation: Automatically generate sitemap.xml for improved SEO.
- Prompt Content: Copy to Clipboard: Allow user to copy the full prompt content with a single click

## Style Guidelines:

- Primary color: Dark, desaturated green (#3A5F0B) to establish the app's primary vibe without overpowering its aesthetic.
- Background color: Very dark desaturated green (#121a06) for the main background.
- Accent color: Slightly brighter, lighter desaturated green (#79895b) for buttons and highlights.
- Body and headline font: 'Inter' sans-serif, providing a clean, modern and easily readable look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, line-based icons for navigation and actions.
- Responsive layout using Tailwind CSS, ensuring optimal viewing experience on all devices.
- Subtle fade-in and transition animations for a smooth user experience.
